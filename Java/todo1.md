## 网络IO、磁盘IO优化
### 网络IO
回答：尽量不走网络、合包

减少网络交互的次数、减少网络传输数据量的大小、尽量减少编码、异步和非阻塞

### 磁盘IO
回答：减少IO次数，ssd等更高效的磁盘

提升I/O性能
增加缓存，减少磁盘访问次数、优化磁盘的管理系统、设计合理的磁盘存储数据块、应用合理的RAID策略提升磁盘I/O
